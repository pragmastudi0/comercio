Dominio

El e-commerce necesita sí o sí un dominio propio para salir a producción. El admin y el punto de venta no lo requieren, pero si hay dominio para la web lo ideal es aprovechar el mismo con subdominios:

› turisteando.com.ar → e-commerce
› admin.turisteando.com.ar → panel de administración
› pos.turisteando.com.ar → punto de venta
Una vez registrado, necesitamos el usuario y contraseña del panel (DonWeb o el registrador que usen) para configurar los DNS y dejar los 3 subdominios apuntando. Pueden completarlo acá o pasárnoslo por canal privado.
